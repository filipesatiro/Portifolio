Portfólio Pessoal - Filipe Satiro

Este é meu portfólio pessoal desenvolvido com HTML, CSS e JavaScript. O projeto apresenta minhas habilidades, formação acadêmica, projetos desenvolvidos e informações de contato profissional.

Visite o site: https://filipesatiro.github.io/Portifolio/

Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

- HTML5: Estrutura semântica da página com seções bem definidas
- CSS3: Estilização completa incluindo variáveis CSS, layout responsivo e tema escuro/claro
- JavaScript: Funcionalidades interativas como alternância de tema e navegação suave
- Font Awesome: Biblioteca de ícones para interface visual

Estrutura do Projeto

O portfólio está organizado da seguinte forma:

index.html
- Estrutura principal da página
- Cabeçalho com navegação fixa
- Seção principal com apresentação pessoal
- Seções de detalhes: vídeo, habilidades, formação e projetos
- Rodapé com informações de contato

estilos/style.css
- Reset CSS e configurações globais
- Variáveis CSS para cores e temas
- Layout responsivo usando CSS Grid
- Estilos para componentes específicos
- Tema escuro/claro com persistência

scripts/script.js
- Sistema de alternância entre tema claro e escuro
- Persistência do tema no localStorage
- Navegação suave entre seções com função personalizada de scroll

imagens/
- Fotos pessoais e imagens dos projetos
- Otimização para web com formatos apropriados

O projeto serve como vitrine das minhas capacidades técnicas e como ponto de partida para futuras colaborações e oportunidades profissionais na área de Tecnologia da Informação.
